# cprop
Constraint Propagation for Binary Programs

## About

A simple, solver independent, Constraint Propagation engine for binary
programs based on the technical report "Nogood Learning for Mixed Integer
Programming", from Sandhol, T. and Shields, R. (2006)
[https://www.cs.cmu.edu/~sandholm/nogoodsForMip.techReport06.pdf].

API is documented
: [here](http://htmlpreview.github.io/?https://github.com/h-g-s/cprop/blob/master/doc/html/cprop_8h.html)
